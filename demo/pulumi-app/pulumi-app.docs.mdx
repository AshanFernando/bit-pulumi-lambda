---

The entry point components to export all Pulumi components in the app. To run the app execute the following command.

```
bit use bit-pulumi-lambda.demo/pulumi-app
bit compile && bit run pulumi-app
```

You can also `cd` to the app directory and execute pulumi commands like `pulumi up` for development.
